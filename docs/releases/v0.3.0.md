# v0.3.0 Release Notes

`v0.3.0` documents the next framework maintenance batch after the `v0.2.0` extraction release.

## Summary

This release improves the public framework shell while keeping commercial workflow logic private. It adds route-level code splitting, formalizes the private workflow adapter boundary, and refreshes the public open source support notes with current community traction.

## Verification

- `npm run check`

## Outcome

- Production entry chunk dropped from about 1,242 kB to about 220 kB.
- Public framework contracts now support private workflow adapter manifests and safety contracts.
- Public documentation now describes how private commercial workflow cores can connect to Tax AI without exposing customer data, credentials or proprietary parser internals.

## Known Limitations

- Tax AI remains a framework prototype, not production tax software.
- Private workflow implementations must remain outside this public repository.
- Additional README screenshots for module-specific views remain tracked by TAI-015.
